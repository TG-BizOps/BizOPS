이슈 유형 구분: 로딩 실패 vs 느린 로딩

이미지 로딩 실패 서버/CDN

  • 이미지 자체를 불러오지 못함
  • 리로드 버튼 노출 (โหลดเว็บตูนอีกครั้ง)
  • 이미지 깨짐/빈 화면/에러 표시
  • 특정 작품·회차에 집중 발생
  • 원인: CDN 캐시 무효화, 이미지 서버 장애, 에셋 누락
  • 대응: 기술팀 에스컬레이션 필요

느린 로딩 / 버퍼링 사용자 환경

  • 이미지가 느리게 로딩됨 (스피너 표시)
  • กำลังโหลดเว็บตูน (웹툰 로딩 중) 상태 지속
  • 서버 지역·사용자 네트워크에 따라 편차
  • 특정 작품에 집중되지 않음
  • 원인: 사용자 대역폭, 지역 CDN 거리, 디바이스 성능
  • 대응: 사용자 가이드(캐시 삭제, 네트워크 전환)
